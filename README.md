# League of Legends Shop
This project was created to test how the LCU api works.
## Install
1. Clone from repo
```
git clone https://github.com/RushWTF/LeagueOfLegends-Shop.git
```
2. Install packages
```
npm install
```
3. Start League of Legends Client
4. Start the script
```
npm start
```
